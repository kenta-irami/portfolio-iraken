# AI Research Idea Generator（個人開発）

![AI Research Idea Generator](/projects/posts/ai-research-generator.png)

ユーザーが入力した研究テーマに対して、**構造化された研究アイデア**を自動生成する Python ツールです。

- **Google Gemini API**による高品質なアイデア生成
- **OpenAlex API**での学術論文検索
- **Pydantic**による構造化データ管理
- **コマンドライン**での簡単操作

学術研究の初期段階を AI 技術でサポートし、研究者の創造性を引き出すツールです。

---

## 背景・課題

研究アイデアの生成と検証には、以下のような課題がありました：

- 既存研究の調査に膨大な時間がかかる
- アイデアの構造化・整理が困難
- 関連論文の検索キーワード選定が難しい
- 仮説・手法・期待結果の体系的整理が必要

これらの課題を、**最新の LLM 技術**と**学術論文 API**の組み合わせで解決するツールを開発しました。

---

## システム構成

### **コア機能**

**1. アイデア生成エンジン（Generator）**

- Google Gemini 2.5 Pro による高品質なアイデア生成
- 英語での内部思考 → 日本語での構造化出力
- JSON Schema による厳密なデータ構造管理

**2. 論文検索エンジン（Searcher）**

- OpenAlex API による学術論文検索
- Polite Pool 利用による安定した API 接続
- Inverted Index 形式のアブストラクト復元処理

**3. データ構造管理**

- Pydantic による型安全なデータモデル
- 研究アイデアの構造化（仮説・手法・期待結果）
- 論文メタデータの統一管理

## 出力される構造化データ

### **研究アイデア**

```python
class ResearchIdea(BaseModel):
    title_en: str          # 英語タイトル（検索用）
    title_ja: str          # 日本語タイトル
    search_keywords: List[str]  # 検索キーワード（3-5個）
    hypothesis: str        # 研究仮説
    methodology: str       # 提案手法
    expected_outcome: str  # 期待される結果
```

### **関連論文情報**

- 論文タイトル・発行年・被引用数
- アブストラクト（最初の 150 文字）
- OpenAlex からの正確なメタデータ

---

## 技術的な特徴

### **API 統合の最適化**

- OpenAlex Polite Pool 利用による高速・安定検索
- Inverted Index 形式のアブストラクト復元アルゴリズム
- エラーハンドリングとレート制限対応

### **データ品質管理**

- Pydantic による入出力データの型検証
- JSON Schema ベースの構造化出力
- 英語キーワード生成による検索精度向上

### **ユーザビリティ**

- シンプルなコマンドライン操作
- 環境変数による設定管理（.env）
- デバッグ情報の適切な表示
