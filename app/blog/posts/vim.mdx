# 小さな会社のための Web サイト制作とミニ自動化

こんにちは。Web制作・Web開発フリーランスの伊良皆 健太です。

このページでは、ぼくが提供している

- 小さな会社・個人事業向けの **Web サイト制作**
- 問い合わせフォームやスプレッドシート連携などの  
  **「ちょっとした業務自動化」**

について、もう少し具体的に紹介します。

---

## 提供していること

### 1. Web サイト制作（メイン）

以下のようなサイトを想定しています。

- 新サービスを紹介するランディングページ（LP）
- 小さな会社・個人事業向けのコーポレートサイト
- ブログ・お知らせ機能付きのシンプルなサイト

ノーコードツール（STUDIO など）から  
Next.js を使ったカスタム実装まで、

- ご予算
- 更新のしやすさ
- 将来の拡張（機能追加や自動化）

を踏まえて、一緒に最適な形を決めていきます。

---

### 2. ちょっとした業務自動化

Web サイト制作がきっかけで見えてくる、
「これ、毎回手作業で大変そうだな」という部分を、
できる範囲で自動化するお手伝いもしています。

たとえば：

- 問い合わせフォーム → Google スプレッドシートに自動保存
- 問い合わせがあったら Slack に自動通知
- 予約や申し込みの一覧を、自動で集計するシート

など、「小さな一部分だけ」の相談でもOKです。

> **ポイント**  
> ・いきなり大きなシステムを作るのではなく、  
> 　まずは「1つの作業をラクにする」ところから始めるイメージです。

---

## どんな方をイメージしているか

- そろそろちゃんとした Web サイトを持ちたい小さな会社さん
- 今のサイトが古くて、スマホ対応も含めて作り直したい方
- 問い合わせ対応や日々の運用が「なんとなく手作業で大変」になってきている方
- IT はあまり得意ではないけれど、少しずつ整えていきたい方

こうした方々と、専門用語をできるだけ減らしつつ、
「いま困っていること」を一緒に整理しながら進めていくスタイルです。

---

## 今後について

今後は、次のようなことも少しずつメニューに加えていく予定です。

- 既存サイトに載せた Q&A やブログ記事を活用した **AI チャットボット**
- 売上・問い合わせ数・予約数などを、一画面で見られる **簡易ダッシュボード**

まずは Web サイト制作から、  
「こんな作業も一緒にラクにできないかな？」という相談をもらえると嬉しいです。
